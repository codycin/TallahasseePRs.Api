# TallahasseePRs

TallahasseePRs is a full‑stack social fitness web application designed to help users track gym progress, share personal records (PRs), and engage with a local fitness‑focused community in Tallahassee. The platform combines social networking features with structured workout and PR tracking, emphasizing clean UI, responsive design, and modern web development best practices.

## Tech Stack

Backend
- C# .NET framework

Database

- PostgreSQL
